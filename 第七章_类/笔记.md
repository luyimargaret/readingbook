# = default含义

# 访问控制与封装
    class和struct的唯一区别就是默认的访问权限不同
## 友元
友元可以访问类的非公有成员
通常把友元的声明和类本身放置在同一个头文件中

# 构造函数值必须初始化的情况
    如果成员时const、引用或者属于未提供默认构造函数的类类型，必须通过构造函数初始值列表为这些成员提供初值。

# 委托构造函数（p292）
委托构造函数使用它所属的类的其他构造函数执行它自己的初始化过程。把自己的一些或者全部职责委托给了其他构造函数。

# 类的静态成员
与类本身相关，不与对象保持关联
类的静态成员存在于任何对象外，对象中不包含任何与静态数据成员有关的数据。
静态成员函数不与任何对象绑定在一起，不包含this指针。静态成员函数不能声明成const的，不能在静态成员函数体内使用this指针。
## 定义静态成员
可以在类外也可以在类内。在类外的话不能重复static关键字。
一般在类的外部定义和初始化每一个静态成员。可以为静态成员提供const整数类型的初始值。
最好把静态数据成员的定义与其他非内联函数定义放在同一个文件夹。
